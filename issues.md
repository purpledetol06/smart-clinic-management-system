# Smart Clinic Management System - User Stories

## Doctor Portal

### User Story 1: Doctor Login
As a doctor
I want to log in to the system with my email and password
So that I can access my patient list and manage appointments

### User Story 2: View Patient List
As a doctor
I want to view all patients assigned to me
So that I can see which patients are registered with me

### User Story 3: View Daily Appointments
As a doctor
I want to view my appointments for each day
So that I can manage my schedule effectively

### User Story 4: Manage Doctor Availability
As a doctor
I want to set my available time slots
So that patients can only book appointments during my available times

## Patient Portal

### User Story 5: Patient Registration
As a patient
I want to register on the system with my personal details
So that I can book appointments with doctors

### User Story 6: Search Doctors
As a patient
I want to search for doctors by speciality and availability
So that I can find the right doctor for my needs

### User Story 7: Book Appointment
As a patient
I want to book an appointment with a doctor
So that I can receive medical consultation

### User Story 8: View My Appointments
As a patient
I want to view my booked appointments
So that I know when to visit the clinic

### User Story 9: Get Prescription
As a patient
I want to receive prescriptions from my doctor
So that I can follow the treatment plan

## Admin Portal

### User Story 10: Admin Login
As an admin
I want to log in to the admin portal
So that I can manage the system

### User Story 11: Add Doctor
As an admin
I want to add new doctors to the system
So that patients can book appointments with them

### User Story 12: Manage Doctor Details
As an admin
I want to update or delete doctor information
So that the doctor list is accurate and up-to-date

### User Story 13: View System Reports
As an admin
I want to view daily appointment reports
So that I can monitor clinic operations

### User Story 14: View Doctor Statistics
As an admin
I want to see which doctor has the most patients
So that I can evaluate doctor performance
